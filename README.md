📊 Data Analysis Agent

An interactive Streamlit-based dashboard designed to simplify Exploratory Data Analysis (EDA) and leverage Artificial Intelligence for data-driven insights. This tool allows users to upload datasets and perform instant statistical analysis, visualizations, and natural language querying.

## 🚀 Key Features

- **Flexible File Upload:** Supports both `.csv` and `.tsv` file formats.
- **Automated EDA:**
  - **Preview:** Quickly scan the top rows of your dataset.
  - **Structure & Info:** View column data types and memory usage.
  - **Statistical Summary:** Generate descriptive statistics (mean, median, etc.) for all features.
  - **Missing Value Detection:** Visualize and list missing data points across the dataset.
- **Interactive Visualizations:** Generate high-quality plots including:
  - Histograms (Distribution Analysis)
  - Boxplots (Outlier Detection)
  - Correlation Heatmaps (Feature Relationship Analysis)
- **AI-Powered Data Chat:** Uses OpenAI's LLMs to answer natural language questions based on the uploaded data's context.
